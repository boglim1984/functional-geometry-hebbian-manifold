# PLY Manifold Artifacts

This directory contains the canonical manifold exports for the Functional Geometry project.

### Core Artifacts
- `phase1_trained_layer4.ply` → trained semantic manifold
- `phase1_untrained_layer4.ply` → random-weight baseline
- `phase2_pixelshuffle_layer4.ply` → pixel-shuffle control

### File Naming Convention
Files follow the `phase_condition_layer.ply` format.

### Storage & Viewing
These files are tracked via **Git LFS**. To view them, use 3D visualization tools such as:
- MeshLab
- CloudCompare
- Blender
